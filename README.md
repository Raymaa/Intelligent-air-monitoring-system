#智能空气监控系统
基于STM32单片机和无线网络技术设计和开发的一款智能空气监测系统，可实时检测室内空气质量，例如：温湿度，亮度，烟雾浓度，PM2.5浓度，并且该系统具有自动报警和风扇多档位变速功能，用户可通过液晶显示屏和微信小程序实现人机交互。
